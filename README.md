# RickAndMortyApp

RickAndMortyApp based on Free API 
* **https://rickandmortyapi.com**

* Architecture - **VIP**
* Layout - **SnapKit**
* Bindings - **Closures**
* Networking - **Alamofire**
* Database - **CoreData**
* Resources - **SDWebImage**

**For Run**
* Download project from repository
* Build project (**CMD+B**)
* Run project (**CMD+R**)
